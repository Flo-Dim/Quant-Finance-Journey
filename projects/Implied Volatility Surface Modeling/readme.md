# Options Analysis Playground

Interactive tool for analyzing options implied volatility and Greeks with AI interpretation.

## Setup
```bash
# Clone
git clone https://github.com/yourusername/options-playground.git
cd options-playground

# Install
pip install -r requirements.txt

# Run
python app.py
```

Open: http://localhost:7860

## Features

- Implied Volatility surfaces
- Greeks (Delta, Gamma, Vega, Theta)
- Term structure analysis
- AI market interpretation (requires Gemini API key)

## AI Setup (Optional)

Get free key: https://makersuite.google.com/app/apikey

Add to :
```python
GEMINI_API_KEY = "your-key-here"
```
